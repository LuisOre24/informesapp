# informesapp
SISTEMA PARA GESTION DE INFORMES MEDICOS (En desarrollo)

Tiene como finalidad facilitar el almacenamiento y la entrega de los informes medicos de los diversos campos que se requiera.



OBJETIVO:

PARA LOS PACIENTES: Facilitar el acceso a los informes de los diversos estudios realizados.

PARA LA ENTIDAD DE SALUD: Ser una herramienta útil el cual facilite el almacenamiento y la búsquda de dichos informes cuando se requiera.
